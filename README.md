## 📌 Overview

- 🧪 Written in native **Batch (.bat)** — no external dependencies
- ⚙️ Designed for **Windows CMD** environment
- 🖱️ Simple terminal UI — clean and responsive
- 🧰 Includes tools for pinging, scanning, resolving DNS, IP tracking, and more
- ✅ Educational and lightweight — no installation needed

---

<h1 align="center">🧭 Features</h1>

### 🌐 IP & Network Tools
- **PINGER** – Ping an IP address
- **IP LOOKUP** – Lookup basic IP info
- **DNS RESOLVER** – Resolve domain names to IP
- **PORT SCANNER** – Scan a target for open ports
- **IP LOGGER** – Generates a Grabify tracking link

### 🔧 Utility & Extra Tools
- **SPAM BOT** – Basic spam bot
- **DOX TOOL** – Compiles publicly available info
- **METHODS** – View list of available methods
- **RAT TOOL GITHUB** – Opens GitHub RAT repo

### 💬 Social & Support
- **DISCORD** – Join the developer's Discord server
- **EXIT** – Closes the tool
